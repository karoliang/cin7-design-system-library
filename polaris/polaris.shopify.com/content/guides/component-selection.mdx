## Data Display Components

### How much data do you need to display?

<BlockStack gap="400">
  <Card>
    <Card.Section>
      <BlockStack gap="300">
        <Badge status="info">Small Dataset (< 50 items)</Badge>
        
        **Use Polaris Components:**
        - `ResourceList` - For product lists, customer lists
        - `DataTable` - For simple tabular data
        - `DescriptionList` - For key-value pairs
        
        ```jsx
        <ResourceList
          items={products}
          renderItem={(item) => (
            <ResourceItem id={item.id}>
              {item.name}
            </ResourceItem>
          )}
        />
        ```
      </BlockStack>
    </Card.Section>
  </Card>
  
  <Card>
    <Card.Section>
      <BlockStack gap="300">
        <Badge status="warning">Medium Dataset (50-1000 items)</Badge>
        
        **Use Polaris with Optimization:**
        - `IndexTable` - For sortable, selectable data
        - `DataTable` with pagination
        - Consider virtual scrolling
        
        ```jsx
        <IndexTable
          resourceName={{singular: 'order', plural: 'orders'}}
          items={orders}
          headings={[
            {title: 'Order'},
            {title: 'Date'},
            {title: 'Customer'},
            {title: 'Total'},
          ]}
          pagination={{
            hasNext: true,
            onNext: loadMore,
          }}
        />
        ```
      </BlockStack>
    </Card.Section>
  </Card>
  
  <Card>
    <Card.Section>
      <BlockStack gap="300">
        <Badge status="critical">Large Dataset (> 1000 items)</Badge>
        
        **Use ExtJS Components:**
        - `ExtDataGrid` - High-performance grid
        - `ExtTreeGrid` - Hierarchical data
        - `ExtPivotGrid` - Data analysis
        
        ```jsx
        <ExtDataGrid
          store={largeDataStore}
          columns={columns}
          features={[
            'virtualScrolling',
            'bufferedRendering',
            'remoteSort',
            'remoteFilter'
          ]}
        />
        ```
      </BlockStack>
    </Card.Section>
  </Card>
</BlockStack>

### Content Organization

<BlockStack gap="400">
  <Card>
    <Card.Section>
      <Text as="h4" variant="headingMd">Cards vs. Sections</Text>
      <BlockStack gap="300">
        <InlineStack gap="400">
          <BlockStack gap="200" inlineAlign="start">
            <Badge>Use Cards when:</Badge>
            <List type="bullet">
              <List.Item>Content is self-contained</List.Item>
              <List.Item>Actions relate to specific content</List.Item>
              <List.Item>Visual separation is needed</List.Item>
            </List>
          </BlockStack>
          
          <BlockStack gap="200" inlineAlign="start">
            <Badge>Use Sections when:</Badge>
            <List type="bullet">
              <List.Item>Content flows together</List.Item>
              <List.Item>Less visual weight needed</List.Item>
              <List.Item>Inside modal or popover</List.Item>
            </List>
          </BlockStack>
        </InlineStack>
      </BlockStack>
    </Card.Section>
  </Card>
</BlockStack>
