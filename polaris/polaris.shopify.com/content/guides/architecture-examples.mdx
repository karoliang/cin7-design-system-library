# Architecture Examples Showcase

<Lede>Explore how Cin7 DSL's multi-layer architecture works in practice with interactive examples for every component.</Lede>

## Overview

Every Polaris component has been enhanced with examples demonstrating our multi-layer architecture:
- **React/Polaris** - Modern component-based UI
- **ExtJS** - Enterprise-grade forms and grids
- **Vanilla JS** - Lightweight DOM manipulation
- **TypeScript** - Type-safe business logic

## How to Use These Examples

1. Navigate to any component page (e.g., [Button](/components/actions/button))
2. Click on the "Examples" section
3. Use the code tabs to switch between implementations:
   - **React** - The default Polaris implementation
   - **HTML** - Rendered HTML output
   - **ExtJS** - Enterprise component version
   - **Vanilla** - Pure JavaScript implementation
   - **TS** - TypeScript with full type definitions
