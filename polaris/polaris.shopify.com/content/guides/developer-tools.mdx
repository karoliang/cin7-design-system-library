# Developer Tools & Setup

<Lede>Set up your development environment with essential tools and extensions for productive Cin7 DSL development.</Lede>

## Quick Setup

```bash
# Install Cin7 DSL CLI globally
npm install -g @cin7/cli

# Create a new project
npx @cin7/create-dsl-app my-app --template=full

# Install recommended VS Code extensions
code --install-extension dbaeumer.vscode-eslint
code --install-extension esbenp.prettier-vscode
code --install-extension bradlc.vscode-tailwindcss
code --install-extension ms-vscode.vscode-typescript-next
```

### VS Code Settings

Create `.vscode/settings.json` in your project:

```json
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "typescript.updateImportsOnFileMove.enabled": "always",
  "typescript.preferences.includePackageJsonAutoImports": "on",
  "emmet.includeLanguages": {
    "javascript": "javascriptreact"
  },
  "tailwindCSS.experimental.classRegex": [
    ["clsx\\(([^)]*)\\)", "(?:'|\"|`)([^']*)(?:'|\"|`)"]
  ]
}
```

### Code Snippets

Create `.vscode/cin7dsl.code-snippets`:

```json
{
  "Cin7 Component": {
    "prefix": "cin7comp",
    "body": [
      "import { ${1:Component} } from '@shopify/polaris';",
      "import { EventBus } from '@cin7/core';",
      "",
      "interface ${TM_FILENAME_BASE}Props {",
      "  ${2:prop}: ${3:type};",
      "}",
      "",
      "export function ${TM_FILENAME_BASE}({ ${2:prop} }: ${TM_FILENAME_BASE}Props) {",
      "  return (",
      "    <${1:Component}>",
      "      $0",
      "    </${1:Component}>",
      "  );",
      "}"
    ]
  },
  "ExtJS Grid": {
    "prefix": "extgrid",
    "body": [
      "import { ExtDataGrid } from '@cin7/extjs-adapters';",
      "",
      "const grid = ExtDataGrid.create({",
      "  store: {",
      "    type: 'json',",
      "    proxy: {",
      "      type: 'ajax',",
      "      url: '${1:/api/data}'",
      "    }",
      "  },",
      "  columns: [",
      "    { text: '${2:Name}', dataIndex: '${3:name}', flex: 1 },",
      "    $0",
      "  ],",
      "  features: ['${4:sorting}', '${5:filtering}']",
      "});"
    ]
  },
  "Repository Pattern": {
    "prefix": "repo",
    "body": [
      "import { Repository, BaseEntity } from '@cin7/typescript-sdk';",
      "",
      "interface ${1:Model} extends BaseEntity {",
      "  ${2:field}: ${3:type};",
      "}",
      "",
      "export class ${1:Model}Repository extends Repository<${1:Model}> {",
      "  protected endpoint = '/${4:models}';",
      "  ",
      "  async findBy${2/(.*)/${1:/capitalize}/}(${2:field}: ${3:type}): Promise<${1:Model}[]> {",
      "    return this.findAll({ ${2:field} });",
      "  }",
      "}"
    ]
  }
}
```

### Console Helpers

Add these to your development environment:

```javascript
// Debug event bus
window.debugEventBus = () => {
  import('@cin7/core').then(({ EventBus }) => {
    EventBus.on('*', (event, data) => {
      console.log(`[EventBus] ${event}:`, data);
    });
  });
};

// Performance monitoring
window.measureComponent = (componentName) => {
  performance.mark(`${componentName}-start`);
  return () => {
    performance.mark(`${componentName}-end`);
    performance.measure(
      componentName,
      `${componentName}-start`,
      `${componentName}-end`
    );
    console.log(performance.getEntriesByName(componentName));
  };
};
```

## CLI Tools

### Cin7 DSL CLI

```bash
# Install globally
npm install -g @cin7/cli

# Available commands
cin7 create <app-name>      # Create new project
cin7 generate <type>        # Generate components/repositories
cin7 migrate                # Migrate from Polaris/ExtJS
cin7 analyze                # Analyze codebase
cin7 add <package>          # Add Cin7 packages
```

### Project Scripts

Add to your `package.json`:

```json
{
  "scripts": {
    "dev": "vite",
    "build": "tsc && vite build",
    "preview": "vite preview",
    "lint": "eslint . --ext ts,tsx --report-unused-disable-directives",
    "format": "prettier --write \"src/**/*.{ts,tsx,js,jsx,css,md}\"",
    "type-check": "tsc --noEmit",
    "test": "vitest",
    "test:ui": "vitest --ui",
    "test:coverage": "vitest --coverage",
    "analyze": "vite-bundle-visualizer",
    "generate:component": "cin7 generate component",
    "generate:repo": "cin7 generate repository",
    "validate": "npm run lint && npm run type-check && npm run test"
  }
}
```

## Development Workflow

### 1. Component Development

```bash
# Generate new component
cin7 generate component MyComponent

# Start development server
npm run dev

# Run tests in watch mode
npm run test:ui
```

### 2. Git Hooks

Install husky for automated checks:

```bash
npm install -D husky lint-staged
npx husky init

# Add pre-commit hook
echo "npx lint-staged" > .husky/pre-commit
```

Configure in `package.json`:

```json
{
  "lint-staged": {
    "*.{ts,tsx}": ["eslint --fix", "prettier --write"],
    "*.{css,md}": "prettier --write"
  }
}
```

### 3. Debugging Setup

#### VS Code Launch Configuration

Create `.vscode/launch.json`:

```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "type": "chrome",
      "request": "launch",
      "name": "Debug in Chrome",
      "url": "http://localhost:5173",
      "webRoot": "${workspaceFolder}/src",
      "sourceMaps": true
    },
    {
      "type": "node",
      "request": "launch",
      "name": "Debug Tests",
      "program": "${workspaceFolder}/node_modules/vitest/vitest.mjs",
      "args": ["--inspect-brk", "--no-coverage"],
      "console": "integratedTerminal",
      "internalConsoleOptions": "neverOpen"
    }
  ]
}
```

## Performance Tools

### Bundle Analysis

```bash
# Analyze bundle size
npm run analyze

# Generate performance report
npx lighthouse http://localhost:5173 --view
```

### Runtime Performance

```javascript
// Add to main.tsx for development
if (import.meta.env.DEV) {
  // React Profiler
  import { Profiler } from 'react';
  
  window.logProfile = (id, phase, actualDuration) => {
    console.log(`⏱️ ${id} (${phase}): ${actualDuration}ms`);
  };
  
  // Wrap components
  <Profiler id="App" onRender={window.logProfile}>
    <App />
  </Profiler>
}
```

## Productivity Tips

### 1. **Use Snippets**
- Install snippet extensions
- Create custom snippets for patterns
- Use Emmet for HTML

### 2. **Keyboard Shortcuts**
- `Cmd+P` - Quick file open
- `Cmd+Shift+P` - Command palette
- `Cmd+B` - Toggle sidebar
- `Cmd+J` - Toggle terminal

### 3. **Multi-Cursor Editing**
- `Alt+Click` - Add cursor
- `Cmd+D` - Select next occurrence
- `Cmd+Shift+L` - Select all occurrences

### 4. **Refactoring**
- `F2` - Rename symbol
- `Cmd+.` - Quick fixes
- `Shift+Alt+O` - Organize imports
