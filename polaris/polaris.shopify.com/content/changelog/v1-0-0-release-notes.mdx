## Technical Highlights

### Type Safety
100% TypeScript coverage with strict mode enabled:
```typescript
interface Component<T = any> {
  get(): T;
  set(value: T): void;
  mount(parent: HTMLElement | string): void;
  unmount(): void;
  destroy(): void;
}
```

### Event-Driven Architecture
Loose coupling via EventBus:
```javascript
EventBus.on('product:updated', (product) => {
  // Handle across all layers
});
```

### Framework Agnostic
Write once, use everywhere:
```javascript
// Vanilla JS
const button = new ButtonComponent({ label: 'Click me' });

// React
<Button>Click me</Button>

// ExtJS
Ext.create('Cin7.Button', { text: 'Click me' });
```

### Performance Optimized
- Lazy loading support
- Tree-shaking friendly
- Minimal bundle sizes
- No runtime dependencies for vanilla JS

## Getting Started

### Installation

```bash
# Create new project
npx @cin7/create-dsl-app my-app --template=full

# Install individual packages
npm install @cin7/core @cin7/vanilla-js
```

### Quick Example

```javascript
import { CheckboxComponent } from '@cin7/vanilla-js';

const checkbox = new CheckboxComponent({
  label: 'I agree to the terms',
  onCheck: () => console.log('User agreed'),
  onUncheck: () => console.log('User disagreed')
});

checkbox.mount('#app');
```
