## Key Concepts

### 1. Layer Selection

Choose the right layer for each part of your application:

- **Vanilla JS**: Lightweight interactions, DOM manipulation
- **React + Polaris**: Modern UI components, state management
- **ExtJS**: Enterprise data grids, complex forms
- **TypeScript SDK**: Business logic, data management

### 2. Shared State

Use the TypeScript SDK patterns to maintain consistent state across layers:

```typescript
import { BaseRepository, BaseUseCase } from '@cin7/typescript-sdk';
import { create } from 'zustand';

// Shared store accessible by all layers
export const useAppStore = create((set) => ({
  // State and actions
}));
```

### 3. Event Bus

Enable communication between layers without tight coupling:

```typescript
import { EventBus } from '@cin7/core/events';

// Emit from one layer
EventBus.emit('data:updated', { id: 123, changes: {...} });

// Listen in another layer
EventBus.on('data:updated', (data) => {
  // Handle update
});
```

### 4. Design Token Consistency

Ensure visual consistency across all layers:

```css
/* Works in all layers */
.my-component {
  background: var(--p-color-bg-surface);
  padding: var(--cin7-space-4);
  border-radius: var(--p-border-radius-300);
}
```

## Best Practices

1. **Start Simple**: Begin with the lightest layer that meets your needs
2. **Progressive Enhancement**: Add complexity only when necessary
3. **Maintain Separation**: Keep business logic in the TypeScript SDK layer
4. **Use Events**: Prefer events over direct coupling between layers
5. **Consistent Styling**: Always use design tokens for styling
