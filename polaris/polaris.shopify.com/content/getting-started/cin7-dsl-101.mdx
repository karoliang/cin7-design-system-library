## What is Cin7 DSL?

Cin7 DSL (Design System Language) is an enterprise-focused design system that combines the elegance of Shopify Polaris with the power of ExtJS. It provides a comprehensive framework for building modern, data-intensive business applications.

The design system is made up of:
- **Polaris components** for modern UI patterns
- **ExtJS integration** for advanced data grids and forms
- **Vanilla JavaScript** utilities for lightweight interactions
- **TypeScript** support for type-safe business logic

## Why Cin7 DSL?

Traditional design systems often fall short when building complex enterprise applications. Cin7 DSL bridges this gap by:

1. **Leveraging Polaris** - Beautiful, accessible, and well-tested UI components
2. **Integrating ExtJS** - Enterprise-grade data management and form controls
3. **Providing flexibility** - Use the right tool for each use case
4. **Ensuring consistency** - Unified design language across all components

## Who is Cin7 DSL for?

Cin7 DSL is designed for:
- **Enterprise developers** building complex business applications
- **UI/UX designers** creating data-intensive interfaces
- **Product teams** needing robust, scalable solutions
- **Organizations** requiring consistent design across applications
