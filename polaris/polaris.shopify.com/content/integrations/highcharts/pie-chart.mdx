# {frontmatter.title}

<Lede>

Pie charts are ideal for showing part-to-whole relationships, composition breakdown, and percentage distribution. Use donut charts for a modern appearance or semi-circle charts for compact displays.

</Lede>

## React Component

```tsx
import { PieChart } from '@cin7/highcharts-adapter/react';

function MarketShareChart() {
  return (
    <PieChart
      title="Market Share by Product"
      subtitle="Q1 2025"
      data={[
        { name: 'Product A', y: 45.0, color: '#5C6AC4' },
        { name: 'Product B', y: 26.8, color: '#006FBB' },
        { name: 'Product C', y: 18.2, color: '#47C1BF' },
        { name: 'Product D', y: 10.0, color: '#955BA5' }
      ]}
      legend={true}
      dataLabels={true}
    />
  );
}
```

## Props

### Required props

| Prop | Type | Description |
|------|------|-------------|
| `data` | `Highcharts.PointOptionsObject[]` | Array of data points with name and value |

### Optional props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Chart title |
| `subtitle` | `string` | - | Chart subtitle |
| `seriesName` | `string` | `'Data'` | Name for the data series |
| `variant` | `'pie' \| 'donut' \| 'semi-circle'` | `'pie'` | Chart style variant |
| `innerSize` | `string \| number` | `'50%'` | Inner size for donut charts |
| `dataLabels` | `boolean` | `true` | Show data labels on slices |
| `legend` | `boolean` | `true` | Show legend |
| `tooltip` | `boolean \| TooltipOptions` | `true` | Configure tooltips |
| `theme` | `Cin7ChartTheme` | `{ mode: 'light' }` | Theme configuration |
| `height` | `number \| string` | `400` | Chart height |
| `width` | `number \| string` | `'100%'` | Chart width |

## ExtJS Integration

```javascript
import { Cin7HighchartsPie } from '@cin7/highcharts-adapter/extjs';

// Register the component
Cin7HighchartsPie.register();

// Create chart panel
const chart = Cin7HighchartsPie.create({
  title: 'Sales Distribution',
  data: [
    { name: 'Online', y: 55 },
    { name: 'Retail', y: 30 },
    { name: 'Wholesale', y: 15 }
  ],
  variant: 'donut',
  dataLabels: true
});
```

## TypeScript SDK

```typescript
import { ChartBuilderFactory } from '@cin7/highcharts-adapter/typescript';

const config = ChartBuilderFactory.createPieChart()
  .setTitle('Customer Segments')
  .setData([
    { name: 'Enterprise', y: 40 },
    { name: 'SMB', y: 35 },
    { name: 'Startup', y: 25 }
  ], 'Customers')
  .asDonut('60%')
  .setLegend(true)
  .build();
```

## Variations

### Standard pie

Traditional pie chart for simple part-to-whole relationships:

```tsx
<PieChart
  title="Budget Allocation"
  data={[
    { name: 'Marketing', y: 35 },
    { name: 'Development', y: 45 },
    { name: 'Operations', y: 20 }
  ]}
/>
```

### Donut chart

Modern appearance with center space for additional context:

```tsx
<PieChart
  variant="donut"
  innerSize="50%"
  data={[
    { name: 'Completed', y: 70 },
    { name: 'In Progress', y: 20 },
    { name: 'Pending', y: 10 }
  ]}
/>
```

### Semi-circle chart

Compact display for gauges or progress indicators:

```tsx
<PieChart
  variant="semi-circle"
  title="Goal Progress"
  data={[
    { name: 'Achieved', y: 75 },
    { name: 'Remaining', y: 25 }
  ]}
/>
```

### With custom colors

Apply brand colors to specific segments:

```tsx
<PieChart
  data={[
    { name: 'Success', y: 80, color: '#108043' },
    { name: 'Warning', y: 15, color: '#FFC453' },
    { name: 'Error', y: 5, color: '#DE3618' }
  ]}
/>
```

### With legend only

Hide data labels and use legend for cleaner appearance:

```tsx
<PieChart
  dataLabels={false}
  legend={true}
  data={[
    { name: 'North America', y: 45 },
    { name: 'Europe', y: 30 },
    { name: 'Asia Pacific', y: 25 }
  ]}
/>
```

### With percentage labels

Show exact percentage values on slices:

```tsx
<PieChart
  dataLabels={true}
  tooltip={{
    pointFormat: '{series.name}: <b>{point.percentage:.1f}%</b>'
  }}
  data={[
    { name: 'Desktop', y: 60 },
    { name: 'Mobile', y: 30 },
    { name: 'Tablet', y: 10 }
  ]}
/>
```

## Best practices

Pie charts should:

- **Show part-to-whole relationships**: Use when parts add up to 100%
- **Limit segments**: Keep to 5-7 slices maximum for clarity
- **Order by size**: Arrange from largest to smallest (clockwise from 12 o'clock)
- **Use distinct colors**: Ensure slices are easily distinguishable
- **Label clearly**: Include both names and percentages

Pie charts should not:

- Compare trends over time (use line charts instead)
- Show negative values (use bar charts instead)
- Display too many categories (causes visual clutter)
- Represent precise value comparisons (use bar charts for accuracy)

## Accessibility

Pie charts include:

- ARIA labels describing the composition
- Keyboard navigation for interactive segments
- High contrast color palettes
- Screen reader-compatible data tables

Provide meaningful `ariaLabel` prop:

```tsx
<PieChart
  ariaLabel="Market share distribution showing Product A leading at 45%, followed by Product B at 26.8%"
  // ... other props
/>
```

## Data transformation

Convert business data to pie chart format:

```typescript
import { DataTransformers } from '@cin7/highcharts-adapter/typescript';

// Category totals
const salesByRegion = [
  { region: 'North America', revenue: 1200 },
  { region: 'Europe', revenue: 980 },
  { region: 'Asia Pacific', revenue: 1450 }
];

const pieData = DataTransformers.transformToPieData(salesByRegion, 'region', 'revenue');

// Calculate percentages
const withPercentages = DataTransformers.calculatePercentage(pieData, 'y');

// Sort by value
const sorted = DataTransformers.sortBy(pieData, 'y', 'desc');
```
